





:class:`FileArrayValue`
=======================


.. py:class:: ansys.tools.variableinterop.file_array_value.FileArrayValue(shape, dtype=float, buffer=None, offset=0, strides=None, order=None)

   Bases: :py:obj:`ansys.tools.variableinterop.variable_value.CommonArrayValue`\ [\ :py:obj:`ansys.tools.variableinterop.file_value.FileValue`\ ]


   
   Stores a value for the ``FileArrayValue`` variable type.

   In Python, the ``FileArrayValue`` type is implemented by extending NumPy's ``ndarray`` type.
   This means that they decay naturally into ``numpy.ndarray`` objects when using NumPy's
   array operators.
