abstract APIStringToValueVisitor.visit_unknown() → None#

Visit the UNKNOWN variable type.

Given that variables of the UNKNOWN type cannot actually be produced, this method always raises NotImplementedError.

Returns:
None

Never returns; always raises NotImplementedError.
